Primer base de datos “CARROS”🚒🚓🚔

Para esta primera base de datos se ingreso al link ... y se fueron seleccionando uno a uno los carros🚓

Dado que daba la opcion para que nosotros eligamos el carro que quisieramos y sea de nuestro agrado😊👩🏻‍💻

A continuacion miraremos los comandos que se pedian digitar, sin embargo en nuestro caso pusimos 3 pantallasos por cada

Uno de ellos dado que algunos salian bastantes resultados.👩🏻‍💻 Primeramente en la parte superior visualizamos todo el Codigo

Comando 1👩🏻‍💻🤫
Cuando el usuario digite este comando indicara únicamente todos los resultados color rojo❤️ en este caso salieron 8 resultados, y como lo nombramos anteriormente solo pondremos 3.

Comando 2🚓
Cuando el usuario digite carros con año de fabricación mayor a 2020 apareceran 9 datos pero en este caso pondremos 3😊

Comando 3👩🏻‍💻
Cuando el usuario digite aparecerán datos mayores o igual precio que pongan😊 ósea los carros que tengan el precio igual o mayor a (Teniendo en cuenta que la pagina decía el precio en dólares)❤️

Comando 4🚓
cuando el usuario digite este comando, aparecerá el carro que tenga capacidad menor de pasajeros al número proporcionado en el código 🚔

Comando 5❤️
Cuando digite este comando aparecerá que la potencia del motor sea menor o igual que el código proporcionado 👩🏻‍💻

Comando 6📲
Cuando el usuario digite este comando aparecerá un carro o varios con la marca diferente a🚔

Comando 7👩🏻‍💻
cuando el usuario digite este comando aparecerán los valores de un carro dentro del rango📲proporcionado de las siguientes marcas.

Comando 8🚔
cuando el usuario digite este comando aparecerán los valores de un carro que no están dentro del rango proporcionado🚔

Comando 9🚒
cuando el usuario digitara esto y aparece un carro o varios con los datos unicos que proporcionamos🚔🤫
